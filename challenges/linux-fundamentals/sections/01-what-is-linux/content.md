## What is Linux?

Linux is a free, open-source operating system kernel created by Linus Torvalds in 1991. Today, Linux powers a vast portion of the modern internet, running on servers, supercomputers, cloud environments, and mobile devices (such as Android).

---

## 1. Operating System vs Kernel

Strictly speaking, **Linux** refers to the operating system kernel itself. A kernel is the core program that manages the system's hardware resources (CPU, memory, storage) and enables other software applications to run.

A complete operating system is built by combining the Linux kernel with utility tools, user space applications, and shell software—most notably those developed by the GNU Project. This is why the full system is often referred to as **GNU/Linux**.

---

## 2. Key Concepts of Linux

If you're coming from Windows or macOS, Linux has a few core philosophical differences:

- **Everything is a File**: In Linux, directories, physical devices (like hard drives or keyboards), and running processes are all represented as files in the filesystem.
- **Multi-user and Multi-tasking**: Linux was designed from the ground up to support multiple users running multiple tasks concurrently with secure isolation.
- **The Power of the CLI**: While desktop Linux distributions offer Graphical User Interfaces (GUIs), the command line interface (CLI) is the industry standard for server administration, automation, and software deployments.
